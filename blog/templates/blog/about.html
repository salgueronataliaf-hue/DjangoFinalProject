Un poco sobre la persona detrás de este blog.

Mi Historia en el Desarrollo Web

Este proyecto nace de mi interés por la programación en **Python** y el *framework* **Django**. Buscaba un entorno robusto para aprender sobre la gestión de usuarios, bases de datos y la creación de un sistema de contenido dinámico. Cada línea de código, desde el modelo `Post` hasta el `LoginRequiredMixin`, ha sido un paso de aprendizaje fundamental.

La idea principal de esta web es funcionar como una plataforma donde cualquier usuario registrado pueda compartir sus pensamientos y páginas, utilizando herramientas modernas como **CKEditor** para texto enriquecido y manteniendo una arquitectura clara basada en el patrón **MVT (Model-View-Template)**.
